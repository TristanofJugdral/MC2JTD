# MC2JTD
------------------------------------------------------------------------------------------------
Hello! Thank you for looking at MC2JTD, a Discord bot that streams chat messages from the CurseForge mod "Cobbleverse" into your Discord servers using the server's chat log. This bot focuses on sending specific information

This bot was created for use in our personal server. Since we ran this server as a multiplayer world, the word "fainting" was censored since the host's message would show up in the Discord channel.  

================ INSTRUCTIONS ================
1. Start Minecraft / server
2. Make sure latest.log exists
3. Run "run_macbook.sh" (MacBook) or "run_windows.bat" (Windows) [SEE BELOW]
4. Keep the terminal open
5. Test in Discord with mc2jtd.help
6. Copy the "logs" file pathway into the bot using mc2jtd.path
7. Test in Minecraft chat or trigger a Cobblemon notification


============== MACBOOK SETUP ==============

In Terminal, write the following: 

cd ~/(*insert folder name containing MC2JTD*)/MC2JTD

bash run_macbook.sh

~-~-~-~-~-~-~-~-~-~-~-~

EXAMPLE: 
cd ~/Downloads/MC2JTD

bash run_macbook.sh

...after this, you can write ./run_macbook.sh in Terminal instead

============== WINDOWS SETUP ==============

Unlike the MacBook file, running the file "run_windows.bat" should be perfectly fine.
